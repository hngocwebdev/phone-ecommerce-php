# Phone E-commerce Website

Website bán điện thoại được xây dựng bằng PHP / Laravel.

## Features
- Product listing
- Shopping cart
- Order management
- Admin dashboard

## Technologies
- PHP / Laravel
- MySQL
- Bootstrap
- HTML / CSS

## Installation

Clone project:

git clone https://github.com/hngocwebdev/phone-ecommerce-php

Install dependencies:

composer install

Run server:

php artisan serve
